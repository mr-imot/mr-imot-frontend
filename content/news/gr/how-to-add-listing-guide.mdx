## Βήμα 1: Βρείτε το κουμπί "Προσθήκη Ακινήτου"

Η προσθήκη αγγελίας γίνεται από το κουμπί "Προσθήκη Ακινήτου", το οποίο είναι ορατό αφού συνδεθείτε με τον λογαριασμό σας. Εναλλακτικά, μπορείτε να χρησιμοποιήσετε το πλευρικό μενού: "Ακίνητα" → "Προσθήκη Ακινήτου".

## Βήμα 2: Συμπληρώστε τα δεδομένα

Συμπληρώστε προσεκτικά όλα τα δεδομένα για το ακίνητό σας.

**ΣΗΜΑΝΤΙΚΟ:** Στο Mister Imot δημοσιεύετε ολόκληρο το έργο. Στην περιγραφή μπορείτε να περιγράψετε λεπτομερώς ποια μεμονωμένα διαμερίσματα ή σπίτια έχετε διαθέσιμα. Με αυτόν τον τρόπο θα αποφύγετε τη μεταφόρτωση πολλαπλών ξεχωριστών αγγελιών και θα μπορείτε εύκολα να επεξεργαστείτε τις πληροφορίες.

Η ιδέα είναι οι αγοραστές να βρουν απευθείας το ακίνητό σας και να επικοινωνήσουν μαζί σας - χωρίς μεσάζοντες, χωρίς προμήθειες.

## Βήμα 3: Τιμή και τοποθεσία

- Η δημοσιευμένη τιμή είναι ανά τετραγωνικό μέτρο
- Αν δεν θέλετε να προσθέσετε τιμή, επιλέξτε την επιλογή "Αίτημα για τιμή"
- Επιλέξτε την ακριβή τοποθεσία στον χάρτη
- Προσθέστε ποιοτικές φωτογραφίες του ακινήτου

## Βήμα 4: Δημοσίευση

Αφού πατήσετε το κουμπί "Δημιουργία Έργου", η αγγελία σας θα είναι ορατή στο πλευρικό μενού "Ακίνητα", όπου μπορείτε να κάνετε επεξεργασίες ανά πάσα στιγμή.

## Πώς να επεξεργαστείτε μια αγγελία;

Η επεξεργασία μιας αγγελίας είναι πολύ εύκολη:

1. **Συνδεθείτε στο προφίλ σας** και μεταβείτε στο πλευρικό μενού "Ακίνητα"
2. **Βρείτε την αγγελία** που θέλετε να επεξεργαστείτε στη λίστα των αγγελιών σας
3. **Κάντε κλικ στην αγγελία** και επιλέξτε την επιλογή "Επεξεργασία"
4. **Κάντε τις απαραίτητες αλλαγές** - μπορείτε να ενημερώσετε:
   - Τίτλο και περιγραφή
   - Τιμή και τοποθεσία
   - Φωτογραφίες
   - Όλες τις άλλες λεπτομέρειες
5. **Αποθηκεύστε τις αλλαγές** - αφού πατήσετε "Αποθήκευση αλλαγών", η αγγελία θα ενημερωθεί αμέσως

Όλες οι αγγελίες σας είναι προσβάσιμες για επεξεργασία ανά πάσα στιγμή από το προφίλ σας. Αυτό σας επιτρέπει να ενημερώνετε τις πληροφορίες όταν έχετε νέα διαθέσιμα διαμερίσματα ή όταν αλλάζετε τις τιμές.
